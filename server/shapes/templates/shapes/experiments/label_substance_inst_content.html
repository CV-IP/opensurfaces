Should I guess if I am unsure?

No. Please choose "I can't tell". We understand that many images are ambiguous.

What do I choose for painted walls and ceilings?

For solid color painted walls, please select "Painted" since it is impossible to know what is below the paint. If the wall is decorated with wallpaper, select "Wallpaper".

What if the laminate looks like tile, granite, or wood?

If a countertop or floor looks like granite, but you think it is actually laminate, select "Granite" anyway. The same applies to other surfaces -- e.g. if it is designed to look like wood or tile, select "Wood" or "Tile" respectively.

What if the wood has varnish?

Finished wood and wood with varnish should be labeled "Wood". If the outermost layer is solid opaque paint (e.g. white paint), select "Painted". Otherwise, select "Wood".

What if there is glass/mirror in front?

Please label the closest surface. If there is glass in front, select "Glass". If the shape is a reflection inside a mirror, select "Mirror".

What do I choose for paintings/posters?

If you can tell that there is glass in front, select "Glass". For oil/latex/acrylic paintings, select "Painted". For posters, select "Paper". If there is any ambiguity, please select "I can't tell" (we would rather that you select this instead of guess).

What about marble tile?

If it is arranged in a regular grid pattern, please select "Tile"

What about painted wood?

If the outermost layer is solid opaque paint (e.g. white paint), select "Painted". Otherwise, select "Wood".

How do I distinguish granite and marble?

LEFT: granite. RIGHT: marble.
Granite has a random texture, while marble has swirly lines. Note that countertops in the US tend to made out of granite rather than marble, since granite is stronger, scratches less, and remains shiny for longer.

Why are we doing this?

We are building a database of materials from images on the web. This is part of ongoing research at Cornell University. Thank you for participating!
