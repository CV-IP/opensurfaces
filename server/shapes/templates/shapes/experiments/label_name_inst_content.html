Your task is to label the name of objects. Each image has a single object (or part of an object) highlighted in red.

If you make a mistake, click the "Go back" button to change your answer.

To better see the highlighted shape, you can drag the image with the mouse or zoom in using the "Zoom in" button.

If you cannot determine the object name, please do not guess. Instead, select one of the options at the bottom of the list:

Not on list
You know the object but it is not on the list.
More than one object
There is more than one object highlighted by the red shape.
I can't tell
The shape is too small or you do not know the object.

Note: If a small portion of an object is highlighted, it is okay to use the name of the entire object.


If the door on the dishwasher is outlined, do you say that it is a "door" or a "dishwasher"?

You should put "dishwasher". The same applies to cabinets and refrigerators.

If the entire floor is covered in carpet, do I select "Rug/mat" or "Floor"?

You should put "Floor". "Rug/mat" refers to removable rugs, not to carpet stapled to the floor. The same applies to carpet stairs -- please put "Stairs".


Why are we doing this?

We are building a database of objects and materials from images on the web. This is part of ongoing research at Cornell University.
