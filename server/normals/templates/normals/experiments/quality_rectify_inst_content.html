For this task, "rectified" means:

  • the blue arrow points away from the surface,
  • the green grid are parallel to edges in the surface, and
  • tile, brick, and wood patterns should be perfectly aligned and the same size throughout the image on the right

If a shape is almost rectified, don't select it. You should not see slanted lines or perspective effects in the image on the right. If you cannot decide on the correct answer, click the button "I can't tell" -- please don't guess.

{% comment %}

Note that many of the images will be wrong; you should expect to only select a few items.

{% endcomment %}

Tip: I find the fastest way to do the task is to do the following: (1) make sure the blue arrow points in the right direction, (2) make sure that patterns on the right are the same size, (3) make sure the grid is parallel to other lines in the image. If any of these are violated, don't click on the image and move on.

Please look at the examples to better understand what "rectified" means.

Important: If you cannot see a red outline around the shape on the left, please try using Google Chrome instead of Firefox.

On the right: the wood patterns are parallel. It should look like you are looking down towards the object.
